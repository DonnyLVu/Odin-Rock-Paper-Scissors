Rock Paper Scissors!

Description

The classic game of rock, paper and of course scissors. The Game is played with 2 people normally but we'll subsitute the second person with thew computer!. Pretty simple description, if you however like to learn more about it click the button for more information.

How to play

The game is pretty simple, in this circumstance the player (you) have the option of choosing between 3 choises: Rock, Paper, and Scissors.
If the player choses the same option as the computer it counts as a tie.

List Of Outcomes

  • Rock beats Scissors
  • Paper beats Rock
  • Scissors beats Paper
  • If the chosen option is the same as the computer, it counts as a tie
